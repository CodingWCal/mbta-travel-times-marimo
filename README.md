# MBTA Travel Times Marimo Notebook

A quick hackathon prototype exploring Boston MBTA travel time data with Marimo notebooks, Python, and open data.

This project was created during a PyData Boston x Cursor Boston evening hackathon. The goal was to experiment with interactive data science workflows and quickly explore patterns in MBTA travel times using a notebook based approach. It was my first time using Marimo Notebook, since prior to this I have only used Jupyter Notebooks.

## Project Overview

This notebook looks at MBTA travel time data and allows basic exploration by month and transit mode.

The current version is an early prototype, not a finished production app. It was built quickly during a live hackathon setting, with the main focus on learning, experimenting, and creating something interactive from public transit data.

## What It Uses

* Python
* Marimo notebooks
* Pandas
* NumPy
* Matplotlib
* MBTA open data

## Main Features

* Loads MBTA travel time CSV data
* Lets the user select a month
* Lets the user choose between heavy rail and light rail data
* Creates simple visualizations for exploring travel time patterns
* Uses Marimo for an interactive notebook experience

## Why I Built This

I wanted to explore how open transit data could be used in a more interactive and visual way.

Since the hackathon focused on data science workflows, this was a good chance to practice working with real world public data, experiment with Marimo notebooks, and build something small but useful around Boston transportation.
